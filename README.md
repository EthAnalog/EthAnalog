- **Tech Stack**: Java, Spring Boot, Spring MVC, MySQL, Apache Tomcat, HTML/CSS/JS  
- **성과**: UX를 고려한 레벨 설계, 즐겨찾기 기반 랭킹 로직 직접 구현
